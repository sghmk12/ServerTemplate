# AI-Real-Estate-Valuator-UI
User interface for https://github.com/svercillo/AI-Real-Estate-Valuation




## RUN FROM SCRATCH:

### Dependencies: 
Yarn --version == *
npm --version == *  
nodejs --version == *

### Commands: 

```
npm install;
npm start;
```


## Deploy:

``` sh
bash prod.sh
```

The following googe apis need to be enabled:

Places API 
Geocoding API 
Maps Javascript API
